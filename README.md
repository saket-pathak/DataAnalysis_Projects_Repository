## 🎯 About This Repository

This repository contains a collection of **Data Analysis Projects** built using:

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SQL (future projects)
* Power BI (future projects)

The projects are organized according to the **Hunter Rank System** from Solo Leveling, where each rank represents a new level of analytical skill and complexity.
