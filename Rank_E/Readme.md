# 🟢 Rank E Problems

> *"Every S-Rank Hunter started as an E-Rank Hunter."*

Welcome to the first stage of the Data Analysis Leveling Journey.

The projects in this folder focus on building the core skills required for every aspiring Data Analyst. These projects emphasize data loading, cleaning, exploration, visualization, and extracting simple but meaningful insights.

---

# 🎯 Objective

At the E-Rank level, the goal is to learn:

* Working with CSV files
* Using Pandas DataFrames
* Basic data cleaning
* Descriptive statistics
* Data visualization
* Insight generation
* Analytical thinking

---

# ⚔️ Skills Unlocked

After completing the projects in this rank, you should be comfortable with:

* `pd.read_csv()`
* `df.head()`
* `df.info()`
* `df.describe()`
* Data filtering
* Sorting data
* Handling missing values
* Creating charts with Matplotlib
* Basic Exploratory Data Analysis (EDA)

---

# 📂 Major Projects

## 1. 🎓 Student Performance Analysis

### Description

Analyze student marks, attendance, and study habits to identify performance patterns.

### Key Questions

* Who are the top performers?
* Does attendance affect marks?
* Does study time improve results?

### Skills Practiced

* Data Cleaning
* Grouping Data
* Statistical Summaries
* Bar Charts

---

## 2. 💰 Personal Expense Tracker Analysis

### Description

Analyze daily expenses and spending habits.

### Key Questions

* Where is most money spent?
* Which category consumes the largest budget?
* How much can be saved monthly?

### Skills Practiced

* Aggregation
* Categorization
* Pie Charts
* Budget Analysis

## 3. 🌦 Weather Data Analysis
